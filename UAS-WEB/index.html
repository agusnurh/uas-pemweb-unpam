Tentang Mata Kuliah

Pemrograman Web 1 adalah mata kuliah fundamental untuk mahasiswa Teknik Informatika yang ingin menguasai teknologi web development.

Mata kuliah ini dirancang untuk memberikan pemahaman mendalam tentang HTML sebagai fondasi pembuatan website. Mahasiswa akan belajar dari konsep dasar hingga dapat membuat halaman web yang terstruktur dengan baik.
