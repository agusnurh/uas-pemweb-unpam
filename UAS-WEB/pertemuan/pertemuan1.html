1. World Wide Web (WWW)

Pengertian WWW

World Wide Web (WWW) adalah kumpulan web di seluruh dunia yang memberikan layanan kepada pengguna untuk mengakses informasi melalui halaman web atau situs yang diinginkan. Internet memberi akses pengguna ke sejumlah besar dokumen yang ditautkan satu sama lain melalui hyperlink.

Konsep Dasar

  • Hypertext: Memungkinkan pengguna menghubungkan kata/frasa untuk mengakses dokumen lain dengan informasi tambahan
  • Hypermedia: Menghubungkan ke objek multimedia seperti gambar, audio, animasi, dan film
  • Client-Server: Internet bekerja dalam format server-client, di mana server menyimpan dan mengirim dokumen, sementara client meminta dokumen tersebut

Komponen WWW

  • HTML (Hypertext Markup Language): Bahasa markup untuk membangun halaman web
  • Browser: Software untuk menampilkan dan mengakses halaman web
  • Web Server: Program yang menyimpan dan mengirim dokumen web
  • HTTP: Protokol komunikasi untuk transfer data di web

2. Cara Kerja Web

Protokol HTTP

HyperText Transmission Protocol (HTTP) adalah protokol komunikasi yang digunakan untuk menghubungkan server ke Internet atau jaringan area lokal (intranet). HTTP bekerja melalui port 80 pada server dan bertanggung jawab untuk:

  • Menghubungkan ke server
  • Mengirim halaman HTML ke browser pengguna
  • Mengunduh file dari server

Alur Proses

  1. User mengetik URL di browser
  2. Browser mengirim request melalui protokol HTTP/TCP-IP
  3. Web server menerima dan memproses request
  4. Server mengirim response berupa HTML
  5. Browser menerjemahkan (interpreting) HTML dan menampilkannya

Catatan Penting

Script HTML dibaca langsung oleh browser dan ditampilkan pada sisi client. Sedangkan script PHP dibaca oleh web server, diterjemahkan oleh PHP Parser, dan hasilnya berupa HTML yang ditampilkan browser.

3. Web Server

Definisi

Web server adalah komputer yang menyimpan file website (HTML, CSS, JavaScript, gambar) dan mengirimkannya ke perangkat pengguna ketika diminta melalui browser.

Fungsi Web Server

  • Menyimpan data: Menyimpan dan mengamankan semua data situs web
  • Menyediakan akses: Memberikan akses ke website yang di-hosting
  • Melayani request: Menerima dan merespon permintaan dari pengguna
  • Bandwidth control: Mengatur lalu lintas jaringan untuk menghindari overload
  • Virtual hosting: Meng-host beberapa website dalam satu server
  • Server-side scripting: Menjalankan bahasa seperti PHP, Python, Ruby, ASP

Keunggulan Apache Web Server

  • Open source dan gratis
  • Dapat dimodifikasi sesuai kebutuhan
  • Mendukung banyak modul tambahan
  • Handal dan stabil
  • Mudah diinstall
  • Perubahan terekam otomatis tanpa restart
  • Cross-platform (Windows, Linux, dll)
  • Dokumentasi lengkap

4. Pengantar HTML

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup untuk menggambarkan struktur halaman web. HTML terdiri dari serangkaian elemen yang meminta browser untuk menampilkan konten dengan cara tertentu.

Kegunaan HTML

  • Menampilkan halaman web beserta konten digital
  • Mengunggah informasi secara online yang dapat diakses kapan saja
  • Membuat form untuk pendaftaran, komentar, dan transaksi
  • Menampilkan multimedia (video, gambar, suara, java applet)

Elemen HTML

Dokumen HTML tersusun dari dua bagian utama:

  • <HEAD>: Berisi title tag, meta tag, CSS code, dan JavaScript code
  • <BODY>: Berisi semua konten yang ditampilkan (teks, hyperlink, gambar, tabel, dll)

5. Struktur Dasar HTML

Berikut adalah struktur dasar dokumen HTML yang lengkap:

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Selamat Datang di HTML</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>

Penjelasan Struktur

  • <!DOCTYPE html> - Deklarasi tipe dokumen HTML5
  • <html> - Root element, pembungkus seluruh dokumen
  • <head> - Berisi informasi meta tentang dokumen
  • <meta charset="UTF-8"> - Menentukan encoding karakter
  • <title> - Judul yang ditampilkan di tab browser
  • <body> - Konten yang ditampilkan di halaman web

6. Tools yang Dibutuhkan

Browser

Browser adalah software client yang memungkinkan pengguna menavigasi dan menampilkan halaman web. Contoh browser:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
  • Opera

Text Editor

Editor adalah software untuk menulis kode HTML. Contoh editor:

  • Visual Studio Code (Recommended)
  • Sublime Text
  • Notepad++
  • Atom
  • Brackets
