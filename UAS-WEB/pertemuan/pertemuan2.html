Basic Tag HTML

Pertemuan 2
35 menit
Dasar

Tujuan Pembelajaran

Setelah menyelesaikan pertemuan ini, mahasiswa mampu:

  • Memahami tag dasar struktur HTML
  • Menggunakan tag HEAD dan elemen-elemennya
  • Menerapkan Meta Tag untuk SEO
  • Mempraktikkan tag format dokumen HTML

1. Tag Dasar Struktur HTML

Pengertian Tag

Tag dalam HTML adalah perintah yang digunakan untuk menandai (markup) elemen dalam dokumen. Tag menggunakan tanda kurung sudut < dan >.

Bentuk-bentuk Tag

a. Tag Berpasangan

Tag yang memiliki pembuka dan penutup:

<nama_elemen> konten </nama_elemen>

Contoh:
<h1>Judul Utama</h1>
<p>Ini adalah paragraf</p>

b. Tag Tunggal

Tag yang tidak memiliki penutup:

<nama_elemen>

Contoh:
<br>
<hr>
<img src="gambar.jpg">

c. Tag dengan Attribut

Tag yang memiliki property tambahan:

<nama_elemen attribut1="nilai1" attribut2="nilai2">konten</nama_elemen>

Contoh:
<p align="center">Teks rata tengah</p>
<img src="foto.jpg" width="300" alt="Foto">

2. Elemen HEAD

Pengertian HEAD

HEAD adalah bagian awal struktur HTML yang berisi informasi meta tentang dokumen. Elemen dalam HEAD tidak ditampilkan di halaman browser, tetapi penting untuk kinerja website.

Fungsi HEAD

  • Menampilkan title pada tab browser
  • Menghubungkan dokumen eksternal (CSS, JavaScript)
  • Menginformasikan keyword untuk search engine
  • Mengatur tampilan dan behavior halaman web

Tag-tag dalam HEAD

a. Tag <TITLE>

Menampilkan judul di tab browser:

<title>Judul Halaman Web</title>

b. Tag <BASE>

Menentukan URL dasar untuk semua link relatif:

<base href="https://www.example.com/direktori/">

c. Tag <LINK>

Menghubungkan dokumen HTML dengan file eksternal:

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">

d. Tag <META>

Menyediakan metadata tentang dokumen HTML:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Deskripsi website">
<meta name="keywords" content="html, css, javascript">
<meta name="author" content="Nama Pembuat">

3. Meta Tag untuk SEO

Meta Description

Memberikan deskripsi singkat tentang konten halaman (maksimal 160-200 karakter):

<meta name="description" content="Belajar HTML dari dasar hingga mahir. Tutorial lengkap dan mudah dipahami untuk pemula.">

Tips Meta Description

Letakkan deskripsi di awal konten halaman. Jika meta description kosong, search engine akan mengambil kalimat pertama dari halaman sebagai deskripsi.

Meta Keywords

Menentukan kata kunci untuk search engine:

<meta name="keywords" content="html, css, javascript, web development">

Peringatan Spam

  • Jangan mengulangi keyword lebih dari 3 kali
  • Jangan gunakan keyword yang tidak relevan dengan konten
  • Contoh SALAH: coklat, coklat kacang, coklat pisang, coklat susu (kata "coklat" berulang lebih dari 3x)

Meta Robots

Mengontrol apakah halaman di-index oleh search engine:

<!-- Index halaman dan ikuti semua link -->
<meta name="robots" content="index, follow">

<!-- Jangan index halaman -->
<meta name="robots" content="noindex, nofollow">

Contoh Lengkap HEAD

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Belajar Meta Tag HTML</title>
    
    <!-- SEO Meta Tags -->
    <meta name="description" content="Panduan lengkap belajar HTML dari dasar">
    <meta name="keywords" content="html, web development, tutorial">
    <meta name="author" content="Nama Pembuat">
    <meta name="robots" content="index, follow">
    
    <!-- External Resources -->
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- Konten di sini -->
</body>
</html>

4. Format Dokumen HTML (BODY)

Tag Formatting Dasar

a. <br> - Line Break

Pindah ke baris baru (seperti menekan Enter):

Baris pertama<br>
Baris kedua<br>
Baris ketiga

b. <p> - Paragraph

Membuat paragraf dengan berbagai alignment:

<p align="left">Rata kiri</p>
<p align="center">Rata tengah</p>
<p align="right">Rata kanan</p>
<p align="justify">Rata kiri-kanan</p>

c. <hr> - Horizontal Rule

Membuat garis horizontal:

<hr>
<hr width="80%" size="5" color="blue" align="center">

d. <h1> - <h6> - Heading

Membuat judul dengan 6 tingkatan (h1 terbesar, h6 terkecil):

<h1>Heading 1 - Terbesar</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6 - Terkecil</h6>

e. <pre> - Preformatted

Menampilkan teks persis seperti yang ditulis (mempertahankan spasi dan enter):

<pre>
Nama        Kelas
======================
Andini      X-A
Budi        X-B
Citra       X-C
</pre>

f. <blockquote> - Block Quote

Membuat teks menjorok ke dalam (indentasi):

<blockquote>
Ini adalah kutipan atau teks yang menjorok ke dalam.
Biasanya digunakan untuk quote atau catatan penting.
</blockquote>

5. Contoh Implementasi

Contoh 1: Website Film

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Website Film Terkenal</title>
</head>
<body>
    <h1>Kumpulan Film Terkenal</h1>
    <hr width="75%" size="3" color="blue" align="left">
    
    <h2>Produksi Sam Spiegel</h2>
    <h3>Jembatan di Atas Sungai Kwai (1957)</h3>
    <p>
        Film ini menampilkan Alec Guinness, William Holden, 
        dan pertunjukkan luar biasa dari Sessue Hayakawa.
    </p>
    
    <h3>Lawrence of Arabia (1962)</h3>
    <p>
        Salah satu film favorit sepanjang masa yang dibintangi 
        Peter O'Toole sebagai sutradara skala besar.
    </p>
</body>
</html>

Contoh 2: Penggunaan Pre

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Penggunaan Pre</title>
</head>
<body bgcolor="cyan">
    <h2>Contoh Penggunaan Heading 2</h2>
    
    <p align="left">Paragraf rata kiri</p>
    <p align="center">Paragraf rata tengah</p>
    <p align="right">Paragraf rata kanan</p>
    
    <pre>
Nama            Email
================================
Andini          andini@gmail.com
Beni            beni@yahoo.com
Kania           kania@yahoo.com
    </pre>
</body>
</html>

Tips Praktis

  • Gunakan <h1> hanya sekali per halaman untuk judul utama
  • Gunakan heading secara hierarkis (h1 → h2 → h3, jangan loncat)
  • Tag <p> otomatis membuat spasi sebelum dan sesudah paragraf
  • Gunakan <pre> untuk menampilkan kode atau data tabel sederhana
  • Meta description sangat penting untuk SEO - buat menarik dan informatif!
