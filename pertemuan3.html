Format Teks & Karakter Khusus

Pertemuan 3
40 menit
Dasar

Tujuan Pembelajaran

Setelah menyelesaikan pertemuan ini, mahasiswa mampu:

  • Menggunakan tag format teks (bold, italic, underline, dll)
  • Mengatur font, size, dan color pada teks
  • Membuat efek marquee untuk teks bergerak
  • Menggunakan karakter khusus (HTML entities)

1. Format Teks HTML

A. Tag Font

Tag <font> digunakan untuk mengatur jenis huruf, warna, dan ukuran teks:

<font face="arial" color="purple" size="4">
    Teks dengan font Arial, warna ungu, ukuran 4
</font>

Attribut Font:

  • face - Jenis huruf (arial, times new roman, courier, dll)
  • color - Warna teks (red, blue, #FF0000, dll)
  • size - Ukuran teks (1-7, atau relatif +1, -1)

Catatan Modern

Tag <font> sudah deprecated di HTML5. Sebaiknya gunakan CSS untuk styling teks. Namun untuk pembelajaran dasar, tag ini masih relevan dipahami.

B. Text Styling Tags

1. Bold (Tebal)

<b>Teks tebal dengan b</b>
<strong>Teks tebal dengan strong</strong>

Perbedaan: <strong> memberikan penekanan semantic (penting), sedangkan <b> hanya visual.

2. Italic (Miring)

<i>Teks miring dengan i</i>
<em>Teks miring dengan em</em>

Perbedaan: <em> memberikan emphasis (penekanan), sedangkan <i> hanya visual.

3. Underline (Garis Bawah)

<u>Teks dengan garis bawah</u>

4. Strike (Coret)

<s>Teks tercoret</s>
<del>Teks yang dihapus</del>

5. Small & Big

<small>Teks lebih kecil dari normal</small>
<big>Teks lebih besar dari normal</big>

6. Subscript & Superscript

H<sub>2</sub>O  <!-- Air: H₂O -->
X<sup>2</sup> + Y<sup>3</sup>  <!-- X² + Y³ -->

C. Kombinasi Format

Tag dapat dikombinasikan, perhatikan urutan penutupan:

<!-- BENAR -->
<b><i><u>Teks tebal miring garis bawah</u></i></b>

<!-- SALAH -->
<b><i><u>Teks</b></i></u>

D. Contoh Implementasi

<p>
    Indonesia merupakan negara <big>kepulauan</big> yang terdiri 
    dari berbagai macam <small>suku bangsa</small> dan 
    <s>adat istiadat</s>.
</p>

<p>
    Rumus Kimia: C<sub>12</sub>H<sub>22</sub>O<sub>11</sub>
</p>

<p>
    Persamaan: X<sub>1</sub><sup>2</sup> + 
    Y<sub>1</sub><sup>3</sup> = 100
</p>

2. Marquee (Teks Bergerak)

Tag <marquee> membuat teks atau objek bergerak secara horizontal atau vertikal:

Attribut Marquee

  • behavior - Cara pergerakan: scroll, slide, alternate
  • direction - Arah: left, right, up, down
  • scrollamount - Kecepatan (angka, default 6)
  • loop - Jumlah perulangan (-1 = infinite)
  • bgcolor - Warna background
  • width - Lebar area marquee
  • height - Tinggi area marquee

Contoh Marquee

<!-- Bergerak ke kiri (default) -->
<marquee>Teks bergerak dari kanan ke kiri</marquee>

<!-- Bergerak ke kanan -->
<marquee direction="right">Teks bergerak ke kanan</marquee>

<!-- Bolak-balik -->
<marquee behavior="alternate">Teks bolak-balik</marquee>

<!-- Dengan kecepatan dan warna -->
<marquee scrollamount="10" bgcolor="yellow">
    Teks cepat dengan background kuning
</marquee>

<!-- Vertikal ke atas -->
<marquee direction="up" height="100">
    Teks bergerak ke atas
</marquee>

Peringatan

Tag <marquee> sudah deprecated dan tidak disarankan untuk website modern karena dapat mengganggu pengalaman pengguna dan aksesibilitas. Gunakan CSS animation sebagai alternatif.

3. Karakter Khusus (HTML Entities)

HTML entities digunakan untuk menampilkan karakter khusus yang tidak bisa langsung diketik atau memiliki fungsi khusus dalam HTML.

Daftar Karakter Khusus Penting

Simbol Deskripsi Entity Code Numeric Code
  Non-breaking space &nbsp; &#160;
< Less than (lebih kecil) &lt; &#60;
> Greater than (lebih besar) &gt; &#62;
& Ampersand &amp; &#38;
" Quotation mark &quot; &#34;
© Copyright &copy; &#169;
® Registered trademark &reg; &#174;
™ Trademark &trade; &#8482;
× Multiplication &times; &#215;
÷ Division &divide; &#247;
« Left angle quote &laquo; &#171;
» Right angle quote &raquo; &#187;

Penggunaan Karakter Khusus

1. Copyright & Trademark

<p>&copy; 2024 Teknik Informatika</p>
<p>Windows&reg; adalah trademark Microsoft</p>
<p>Brand X&trade;</p>

Hasil: © 2024 Teknik Informatika | Windows® adalah trademark Microsoft | Brand X™

2. Operator Matematika

<p>5 &times; 3 = 15</p>
<p>20 &divide; 4 = 5</p>

Hasil: 5 × 3 = 15 | 20 ÷ 4 = 5

3. Menampilkan Tag HTML

<p>Gunakan tag &lt;b&gt; untuk teks tebal</p>

Hasil: Gunakan tag <b> untuk teks tebal

4. Non-Breaking Space

<p>Harga: Rp&nbsp;100.000</p>

Fungsi: Mencegah pemisahan teks "Rp" dan "100.000" ke baris berbeda.

Code Playground - Latihan Format Teks

Tips Praktis

  • Gunakan <strong> dan <em> daripada <b> dan <i> untuk semantic HTML
  • HTML entities sangat penting saat menampilkan kode HTML dalam halaman
  • &nbsp; berguna untuk membuat spasi yang tidak akan di-break oleh browser
  • Hindari penggunaan marquee di website production - gunakan CSS animation
  • Selalu tutup tag dengan urutan yang benar (LIFO - Last In First Out)

4. Contoh Lengkap

<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Format Teks Lengkap</title>
</head>
<body>
    <h1>Contoh Lengkap Format Teks</h1>
    
    <!-- Font Styling -->
    <p>
        <font face="arial" color="blue" size="5">
            Teks dengan font Arial, biru, ukuran 5
        </font>
    </p>
    
    <!-- Text Formatting -->
    <p>
        <strong>Teks Penting</strong> dan <em>Teks Ditekankan</em>
    </p>
    
    <!-- Subscript & Superscript -->
    <p>
        Rumus: E = mc<sup>2</sup><br>
        Molekul: H<sub>2</sub>SO<sub>4</sub>
    </p>
    
    <!-- HTML Entities -->
    <p>
        &copy; 2024 Universitas Pamulang<br>
        Harga: Rp&nbsp;100.000<br>
        10 &times; 5 = 50
    </p>
    
    <!-- Marquee -->
    <marquee behavior="alternate" scrollamount="5">
        Selamat Datang di Website Kami!
    </marquee>
</body>
</html>
